# 🎨 Change of Shape Characteristics with JavaFX

This project is a JavaFX application that allows users to interactively explore and modify different geometric shapes.

The application provides a simple graphical interface where users can choose between multiple shapes and dynamically change their properties such as angles, width, and height.

---

## 🚀 Features

### 🔷 Supported Shapes
The application supports 8 different JavaFX shapes:
- Arc  
- Cubic Curve  
- Cylinder  
- Ellipse  
- Line  
- Polygon  
- Rectangle  


---

## ⚙️ Functionality

- Select one of the available shapes
- Modify:
  - Width
  - Height
  - Angles
- Real-time visual updates of the shape
- Interactive UI for experimenting with geometry

---

## 💡 Purpose of the Project

This project was created to:
- Practice **JavaFX development**
- Understand how graphical shapes work
- Learn how to manipulate shape properties dynamically
- Explore UI interaction with graphical objects

---

## 🛠️ Tech Stack

- Java
- JavaFX
- Scene Builder (optional for UI design)

---

## 📌 Notes

This application demonstrates how JavaFX handles shapes using object-oriented design, where each shape is a class with adjustable properties such as size, position, and angles.
